#Instructions of setting up

1. Set up a PaperMC server. Instructions can be found here: https://paper.readthedocs.io/en/latest/

2. Download the latest stable build of citizens from https://ci.citizensnpcs.co/job/Citizens2 and place it into YourServerFolder\plugins
Last build we've tested is #2220

3. Download the latest stable build of Denizen from https://ci.citizensnpcs.co/job/Denizen/ and place it into YourServerFolder\plugins
Last build we've tested is #1723

4. Copy the folder this file is into YourServerFolder\plugins\Denizen
